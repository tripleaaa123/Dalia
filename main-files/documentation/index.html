Layout

Margins and paddings

Control the padding on one side of an element using the p{t|r|b|l}-{size} utilities.

All examples also works with paddings.

            
              .mt20 { margin-top: 20px; }
              .mt25 { margin-top: 25px; }
              .mt30 { margin-top: 30px; }
              .mt35 { margin-top: 35px; }
              .mt40 { margin-top: 40px; }
              .mt45 { margin-top: 45px; }
              .mt50 { margin-top: 50px; }
              .mt55 { margin-top: 55px; }
              .mt60 { margin-top: 60px; }
              .mt65 { margin-top: 65px; }
              .mt70 { margin-top: 70px; }
              .mt75 { margin-top: 75px; }
              .mt80 { margin-top: 80px; }

Horizontal and vertical spacing

Control horizontal and vertical spacing of an element using the mx-{size} utilities.

            
              <div class="mx-10"></div>
              <div class="py-10"></div>

Responsive

To control the margin of an element at a specific breakpoint, add a {screen}: prefix to any existing margin utility. For example, adding the class mt20-md to an element would apply the mt20 utility at medium screen sizes and below.

            
              <div class="mt-30 mt20-md mt20-sm"></div>
